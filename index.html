Development is now reset.

Old codes are too messy and low efficiency. The whole code base is now rebuild. In the meanwhile, many models are now deprecated for they are either built with 3D Builder or have too may unnecessary faces.

Overview

This game is based on djkaktus's SCP-1730, powered by Unity3D. A totally fan made fanart.

It contains a lot of Easter Eggs (With strongly subjective style).

The design is affected by Halo series and SCP - Containment Breach.

Story

Player mainly acts an original character named John Smith , a Level-3 researcher working in Site-13 . Player will explore and survive in Site-13 after a serious containment breach until Tau-5 rescue you.

Also, there are some other story branch you can explore different end rather than the true end.

Goals

Here lists some goals that current development is working on.

  • Modular scene to make single map much larger.
  • More weapons.
  • Detailed models.
  • Rebuild code base to make the entire game logic more efficient.
  • Rebuild external tools with .Net 5 and Avalonia to make them cross-platform.
  • More real-time CG animations.
  • At least 2 controllable character.
  • Clever AI than NavMeshAgent.

    • Note: Goals may change at any time.

    System Requirements

    Minimal System Requirement

  • CPU: Intel(R) Core(TM) i7-8650U or equivalent
  • GPU: GTX 1060 Mobile or equivalent
  • Memory: 8GB
  • OS: Windows 10 or newer/ Debian 10 or newer
  • Input: Generic Keyboard and Mouse
  • DirectX: 11 or newer
  • OpenGL: Untested
  • To use external tools, you may need .Net Framework 4.5

    • Note: Cannot ensure 1080P 60FPS in this combination.
