What's this?

This namespace contains ths most codes about game logic in levels of Site13Kernel.

Understanding Codes

ResourceBuilder

This class is designed to prepare resources (Prefabs, mostly) that current level will use.

The basic logic is to serialize String-GameObject, Int-GameObject and String-Int lists to dictionaries.

Remarks

  • isMain - this builder is the main resource, will erase previous converted resource dictionaries.
  • isSub - this builder is a sub-builder, will merge its converted resource dictionaries into main dictionaries.
  • isControlled - use Init() instead of Start().
